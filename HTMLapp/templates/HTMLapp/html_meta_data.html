HTML Meta Data

Previous Next

Metadata are used tell the browsers how to display content or reload page, they specify the character set and viewport settings for the browser, they are also used to tell the search engine about the page description, keywords and author of the document.

Metadata is specified using the <meta> tag.

Types of metadata


Charset meta

It defines the type of character set to be used by the browser

<meta charset="UTF-8">

Document refresh meta

It tells the browser when to refresh a web page

The content="50" part tells the browser to refresh after every 50 seconds.

<meta http-equiv="refresh" content="50">

Viewport meta

It instructs the browser on how to control the page's dimensions and scaling.

The width=device-width part sets the width of the page to follow the screen-width of the device (This will vary depending on the device).

The initial-scale=1.0 part sets the initial zoom level when the page is first loaded by the browser.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Keywords meta

It defines the keywords of a web page to search engines

<meta name="keywords" content="HTML, AJAX">

Description meta

It defines the description of a web page

<meta name="description" content="Best Programming site">

Author meta

It defines the author of a web page

<meta name="author" content="Francis">

Example of a web page with all the above metas

<meta charset="UTF-8">
<meta name="description" content="Best Programming site">
<meta name="keywords" content="HTML, AJAX">
<meta name="author" content="Francis">
