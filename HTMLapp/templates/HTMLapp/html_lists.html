HTML Lists are used to group a set of related iteams.

Types of HTML lists

  • Ordered Lists
  • Unordered Lists
  • Description Lists

Ordered Lists

Ordered lists are numbered in numerical or alphabetical way. They are defined by the <ol> tag

Ordered list style type attribute

  1. type="1" - This will display a list with numbers(1,2,3,4...). This is the default list style type in ordered lists.
    2. Example code
    <ol type="1">
        <li>Joseph</li>
        <li>John</li>
        <li>Biden</li>
    </ol>
    The output
    1. Joseph
    2. John
    3. Biden

  2. type="A" - This will display a list with uppercase letters(A,B,C,D...)
    3. Example code
    <ol type="A">
        <li>Joseph</li>
        <li>John</li>
        <li>Biden</li>
    </ol>
    The output
    1. Joseph
    2. John
    3. Biden

  3. type="a" - This will display a list with lowercase letters(a,b,c,d...)
    4. Example code
    <ol type="a">
        <li>Joseph</li>
        <li>John</li>
        <li>Biden</li>
    </ol>
    The output
    1. Joseph
    2. John
    3. Biden

  4. type="I" - This will display a list with roman numbers in uppercase(I,II,III,IV...)
    5. Example code
    <ol type="I">
        <li>Joseph</li>
        <li>John</li>
        <li>Biden</li>
    </ol>
    The output
    1. Joseph
    2. John
    3. Biden

  5. type="i" - This will display a list with roman numbers in uppercase(i,i,i,iv...)
    6. Example code
    <ol type="i">
        <li>Joseph</li>
        <li>John</li>
        <li>Biden</li>
    </ol>
    The output
    1. Joseph
    2. John
    3. Biden

Unordered Lists

Unordered lists are defined by <ul> tag. They are marked with circle, square or disc markers.
The marker can be changed by changing the "list-style-type" property.

Types of unordered list markers

  1. Circle
    2. Example code
    <ul style="list-style-type: circle;">
        <li>Joseph</li>
        <li>John</li>
        <li>Biden</li>
    </ul>
    The output
    • Joseph
    • John
    • Biden

  2. Disc
    3. Example code
    <ul style="list-style-type: disc;">
        <li>Joseph</li>
        <li>John</li>
        <li>Biden</li>
    </ul>
    The output
    • Joseph
    • John
    • Biden

  3. Square
    4. Example code
    <ul style="list-style-type: square;">
        <li>Joseph</li>
        <li>John</li>
        <li>Biden</li>
    </ul>
    The output
    • Joseph
    • John
    • Biden

  4. None
    5. Example code
    <ul style="list-style-type: none;">
        <li>Joseph</li>
        <li>John</li>
        <li>Biden</li>
    </ul>
    The output
    • Joseph
    • John
    • Biden

Control the start of Lists

You can Control the start of a list by defining the start attribute

Example code
<ol start="10">
    <li>Joseph</li>
    <li>John</li>
    <li>Biden</li>
</ol>
The output
  1. Joseph
  2. John
  3. Biden

Nested HTML Lists

Nested lists are used when we want to list a list inside a list.

Example code
<ol>
    <li>Joseph</li>
    <li>John</li>
    <ul>
        <li>Joseph</li>
        <li>John</li>
    <ul>
    <li>Biden</li>
</ol>
The output
  1. Joseph
  2. John
    • Doe
    • Elon
  3. Biden

Descriptive HTML Lists

Descriptive lists are used list iteams with a description of each of them.

Example code
<dl>
    <dt>HTML</li>
    <dd>Hyper Text Markup Language</li>
    <dt>CSS</li>
    <dd>- Cascading Style Sheets</li>
</dl>
The output:
HTML
- Hyper Text Markup Language
CSS
- Cascading Style Sheets
