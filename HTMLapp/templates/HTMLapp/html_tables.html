HTML Tables

Previous Next

HTML Tables are templates for arranging data into rows and columns in wed development. HTML tables are defined by <table>tag.

HTML Table Tags

  • <table> - This is defines the HTML table.
  • <th> - This defines the head of the HTML table.
  • <tr> - This defines a row in a table.
  • <td> - This defines the table data.

Example of a basic HTML table code


<table style="width:100%">
  <tr>
    <th>Name</th>
    <th>Country</th>
    <th>Gender</th>
  </tr>
  <tr>
    <td>John</td>
    <td>North Korea</td>
    <td>Male</td>
  </tr>
  <tr>
    <td>Doe</td>
    <td>USA</td>
    <td>Male</td>
  </tr>
  <tr>
    <td>Marry</td>
    <td>Canada</td>
    <td>Female</td>
  </tr>
</table>

Output:
Name Country Gender
John North Korea Male
Doe USA Male
Marry Canada Female

HTML Table with stripped rows

Add the following CSS code to display a stripped table

<style>
table {
    border-collapse: collapse;
    width: 100%;
}

td, th {
    border: 1px solid #f2f2f2;
    text-align: left;
    padding: 8px;
}

tr:nth-child(even) {
    background-color: #f1f1f1;
}
</style>

Output:
Name Country Gender
John North Korea Male
Doe USA Male
Marry Canada Female

HTML table column span

This allows a cell to span more than one column.

<table style="width:100%">
  <tr>
    <th colspan="2">Name</th>
    <th>Gender</th>
  </tr>
  <tr>
    <td>John</td>
    <td>Doe</td>
    <td>Male</td>
  </tr>
  <tr>
    <td>Biden</td>
    <td>Hawker</td>
    <td>Male</td>
  </tr>
  <tr>
    <td>Marry</td>
    <td>Keziah</td>
    <td>Female</td>
  </tr>
</table>

Output:
Name Gender
John Doe Male
Biden Hawker Male
Marry Keziah Female

HTML table row span

This allows a cell to span more than one row.

<table style="width:100%">
  <tr>
    <th>Name</th>
    <td>Elon Musk</td>
  </tr>
  <tr>
    <th rowspan="2">Companies</th>
    <td>Tesla</td>
</tr>
  <tr>
    <td>Boring Company</td>
  </tr>
</table>

Output:
Name: Elon Musk
Companies: Tesla
SpaceX
Boring Company
