Defining HTML color using RGB values

RGB represents RED, GREEN and BLUE. The colors are represented using values that ranges from 0 - 250, for example rgb(255, 255, 255).

The values 0 - 255 define the intensity of the color. If the value of a color is set to 0 then the color will not be displayed. If the values for all the three color is set to 0 the a black color will be displayed.

Defining HTML color using RGBA values

RGBA is an extention of RGB, It has an addational value of alpha parameter that specifies the opacity of the color i.e rgba(red, green, blue, alpha). Alpha ranges from 0 to 1.0

Example code to display HTML colors using RGBA values

<span style="background-color: rgb(155, 255, 50, 0.2)"> rgb(155, 255, 50, 0.2)</span>
<span style="background-color: rgb(155, 255, 50, 0.4)"> rgb(155, 255, 50, 0.4)</span>
<span style="background-color: rgb(155, 255, 50, 0.6)"> rgb(155, 255, 50, 0.6)</span>
<span style="background-color: rgb(155, 255, 50, 0.8)"> rgb(155, 255, 50, 0.8)</span>

Output of using RGBA colors.

rgba(155, 255, 50, 0.2 ) rgba(155, 255, 50, 0.4 ) rgba(155, 255, 50, 0.6 ) rgba(155, 255, 50, 0.8 )

Defining HTML color using HSL values

HSL stands for Hue, Saturation and Lightness.

Hue

It is the degree in which all the colors ranges, Which is normally from 0 to 360. 0 represents red, 120 represents green, and 240 represents blue.

Example code to display HTML colors using HSL with varying Hue value

<span style="background-color: hsl(0, 50%, 50%)"> hsl(0, 50%, 50%)</span>
<span style="background-color: hsl(120, 50%, 50%)"> hsl(120, 50%, 50%)</span>
<span style="background-color: hsl(240, 50%, 50%)"> hsl(240, 50%, 50%)</span>
<span style="background-color: hsl(320, 50%, 50%)"> hsl(320, 50%, 50%)</span>

Output showing variation of Hue value in HTML colors

hsl( 0 , 50%, 50%) hsl( 120 , 50%, 50%) hsl( 240 , 50%, 50%) hsl( 320, 50%, 50%)

Saturation

It is a percentage value that ranges from 0% to 100% and describes the intensity of the color.

Example code to display HTML colors using HSL with varying Saturation value

<span style="background-color: hsl(0, 20%, 50%)"> hsl(0, 20%, 50%)</span>
<span style="background-color: hsl(0, 40%, 50%)"> hsl(0, 40%, 50%)</span>
<span style="background-color: hsl(0, 60%, 50%)"> hsl(0, 60%, 50%)</span>
<span style="background-color: hsl(0, 80%, 50%)"> hsl(0, 80%, 50%)</span>

Output showing variation of Saturation

hsl(0, 20% , 50%) hsl(0, 40% , 50%) hsl(0, 60% , 50%) hsl(0, 80% , 50%)

Lightness

It is a percentage value that describes how much light a color should have, Ussually ranges from 0% to 100%. The 0% is black, and 100% is white.

Example code to display HTML colors using HSL with varying Lightness value

<span style="background-color: hsl(120, 50%, 20%)"> hsl(120, 50%, 20%)</span>
<span style="background-color: hsl(120, 50%, 40%)"> hsl(120, 50%, 40%)</span>
<span style="background-color: hsl(120, 50%, 60%)"> hsl(120, 50%, 60%)</span>
<span style="background-color: hsl(120, 50%, 80%)"> hsl(120, 50%, 80%)</span>

Output showing variation of Lightness

hsl(120, 50%, 20% ) hsl(120, 50%, 40% ) hsl(120, 50%, 60% ) hsl(120, 50%, 80% )

Defining HTML color using HSLA values

HSLA is an extention of HSL, It has an addational value of alpha parameter that specifies the level of transparency the color should have i.e hsla(hue, saturation, lightness, alpha). Alpha ranges from 0 to 1.0

Example code to display HTML colors using HSLA with varying alpha value

<span style="background-color: hsl(0, 100%, 50%, 0.1)"> hsl(0, 100%, 50%, 0.1)</span>
<span style="background-color: hsl(0, 100%, 50%, 0.4)"> hsl(0, 100%, 50%, 0.4)</span>
<span style="background-color: hsl(0, 100%, 50%, 0.6)"> hsl(0, 100%, 50%, 0.5)</span>
<span style="background-color: hsl(0, 100%, 50%, 0.8)"> hsl(0, 100%, 50%, 0.8)</span>

Output of using HSLA colors.

hsla(0, 100%, 50%, 0.1 ) hsla(0, 100%, 50%, 0.4 ) hsla(0, 100%, 50%, 0.6 ) hsla(0, 100%, 50%, 0.8 )

Defining HTML color using HEX values

This is where a color is specified with hexadecimal values that range form 00 to ff (#RRGGBB) , where the RR is Red, GG is Green and BB is Blue. The range 00 - ff is the same as (0 - 250).

Example code to display HTML colors using HEX values

<span style="background-color: #f1f1f1"> #f1f1f1</span>
<span style="background-color: #aaddff"> #aaddff</span>
<span style="background-color: #ababab"> #ababab</span>
<span style="background-color: #2343ad"> #2343ad</span>

Output of using Hex colors.

#f1f1f1 #aaddff #ababab #2343ad
