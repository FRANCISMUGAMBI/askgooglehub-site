HTML Image tag

Previous Next

HTML image is defined using the <img> tag.

The sytax for defining an image in a web page is: <img src="image url" alt="alternative Text" width="image width" height="image height" >

HTML Image Attributes

src attribute

It specifies the path to the image e.g (src="mypic.png")

alt attribute

It specifies the alternative text that will be displayed if the the image fails to load e.g (alt="tree imge")

width attribute

It specifies the width of the image e.g (width="400px")

height attribute

It specifies the height of the image e.g (height="250px")

Example

<img src="audi.jpg" alt="Audi car" width="100px" height="300px">


Output



Using Image as a link

Images can be used as a link by putting the image tag inside the <a> tag.

Example showing using image as a link

<a href="/">
<img src="audi.jpg" alt="Audi car" width="100px" height="300px">
</a>


Output: Click the image to navigate to homepage
