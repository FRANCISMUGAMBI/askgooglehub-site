How to write HTML comments

Previous Next

HTML comments are not displayed in the browser, but they can help document your HTML source code. They can also be used for debugging to exclude an html code from being displayed by the browser. When commenting in HTML code we use <!-- to show the start of the comment and close it widh --> to show the end of the comment. Example:

<!-- I am a comment in HTML code--> - This is a comment in HTML code. It cannot be displayed in a browser.

Example of an HTML comment that prevents the code from being displayed by the browser.

<!-- This is the start of the comment.
<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>
<p> My first paragraph.</p>

</body>
</html>
--> This is the end of the comment
