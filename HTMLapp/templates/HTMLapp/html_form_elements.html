HTML forms are used to collect the user input that is sent to the server for processing. Forms are defined using the <form> tag.

HTML Form Elements

  • <form> - It defines an HTML form. It creates a HTML form.
  • <input> - It is used to get an input from the user.
  • <textarea> - It defines a label for the form elements.
  • <button> - It defines a button.
  • <fieldset> - It groups related elements in a form.
  • <option> - It defines an option to select from a group.
  • <Label> - It defines a label for the input element.
  • <legend> - It adds a caption to a fieldset.
  • <optgroup> - It shows the related options in a list.

HTML <textarea> Form Element

The HTML <textarea> is useful when an input of large text is required from the user.
The rows attribute is used to indicate the number of visible lines to be shown in a textarea while the cols attribute is used to indicate the visible width of a textarea.

Example
<label>Last Name </label>
<textarea cols="30" rows="5" >This is a textarea </textarea>

HTML <select> Form Element

The HTML <select> element is used to define a dropdown list.
The HTML <option> element defines an option that is to be selected that is an iteam of the dropdown list.

The HTML <select> element attributes.

selected - This defines the iteam that is to be selected in default.
size - This defines the number of visible lines from the dropdown list.
multiple - This allows the user to select more than one value.

Example
<label >Choose your favourite fruit(s): </label>
<select size ="3" multiple >
  <option value="orange">Orange</option>
  <option value="mango" selected >Mango</option>
  <option value="pineapple">Pineapple</option>
  <option value="watermelon">Watermelon</option>
</select>

Output:



HTML <datalist> Form Element
