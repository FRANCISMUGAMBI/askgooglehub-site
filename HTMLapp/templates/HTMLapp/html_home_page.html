HTML(Hypertext Markup Language) is the standard markup language for creating Web pages that are to be displayed in a web browser. This can be a web application page or a website page.

The history of HTML

HTML was first written by Tim Berners-Lee in the year 1993. He was working at CERN (Conseil Européen pour la Recherche Nucléaire; European Organization for Nuclear Research) It was created for the distribution of relatively simple structured documents suitable for use by authors who were scientists or academics rather than experts in printing or other aspects of document formatting. Since then HTML have been undergoing some development changes and more features have been added to it. HTML describes the structure of a web page semantically. The Web browsers receive HTML documents from web server or from local storage and render the documents into multimedia web pages.

Fuctions of HTML

  • It is the standard markup language used for creating Web pages
  • It is used to tell the browser how to display the content
  • It describes the structure of a Web page
  • It labels pieces of content using its elements
