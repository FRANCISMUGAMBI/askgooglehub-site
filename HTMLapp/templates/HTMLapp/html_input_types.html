HTML Input Types

  • <input type="text"> - It defines a text input field.
  • <input type="email"> - It defines an email input field.
  • <input type="password"> - It defines a password input field.
  • <input type="checkbox"> - It defines a checkbox.
  • <input type="number"> - It defines a number input field.
  • <input type="radio"> - It defines a raddio button.
  • <input type="submit"> - It defines a submit button.
  • <input type="button"> It defines a button.
  • <input type="reset"> - It defines a reset button.
  • <input type="image"> - It defines an image field.
  • <input type="tel"> - It defines a telephone input field.
  • <input type="url"> - It defines a url input field.
  • <input type="file"> It defines a file field. This can be an image or a video.
  • <input type="date"> - It defines a date input field.
  • <input type="datetime-local"> - It defines a date input field that collects the date and time.
  • <input type="week"> - It defines an input field that collects data of the week in a year
  • <input type="month"> - It defines an input field for a month in a year.
  • <input type="search"> - It defines a search field.
  • <input type="range"> - It defines a range input
  • <input type="hidden"> - It defines a hidden field.
  • <input type="color"> - It defines a color input field.
