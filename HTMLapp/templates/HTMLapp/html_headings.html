Description of HTML headings H1 to H6

  • <H1> - This is the main heading of a web page, each page should have only one H1 heading
  • <H2> - This is the heading that follows after the H1 heading, it can be used to define sub-topics of an HTML page.
  • <H3> - H3 heading can be used after H2 tag to create a sub-topic relating to the topic of H2 heading .
  • <H4> - It is defined after the H3 heading
  • <H5> - It is defined after H4 heading
  • <H6> - This is the smallest heading of a web page.
NB/ The headings should not be used to enlarge the size of a text instead use the font-size CSS property, they should only be used for creating heading of topics and sub-topics.
