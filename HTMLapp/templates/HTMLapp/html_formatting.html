What is HTML formatting?

HTML formatting is a method for defining text with special meaning, for example making a text to be italic or bold.

HTML Formatting Elements

  • <bold> - It bolds the text.
  • <small> - It displays a small text.
  • <strong> - Used when displaying an important text by making it bold.
  • <i> - Italizes the text.
  • <del> - Used to show that the text have been deleted by displaying a line across the text.
  • <sub> - Used when writting a subscript.
  • <sup> - Used when writting a superscript.
  • <ins> - Used to define an inserted text.
  • <em> - Used to to show emphasize of a text.
  • <mark> - Used to show a marked text by italizing it.

HTML bold element (<bold>)

Example of an HTML bold text code
<b>This is a bold text</b>
<b>I am a bold text</b>
Output:
This is a bold text
I am a bold text
